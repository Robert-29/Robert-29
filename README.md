### 🚀 Sobre mí

Me apasiona construir sistemas completos: desde el hardware (microcontroladores, sensores, servomotores) hasta plataformas web full-stack con inteligencia artificial integrada. Trabajo con tecnologías de frontend, backend, bases de datos relacionales, redes y sistemas embebidos.

- 🔭 Actualmente desarrollando **SIBIA**, un sistema inteligente de bienestar académico universitario con IA explicable (XAI)
- 🌉 Construí un sistema de automatización de un puente levadizo con **ESP32** y dashboard web en tiempo real
- 🌱 Explorando constantemente: redes (CCNA), desarrollo móvil, y arquitecturas de AI aplicada
- 🐱 Cuando no programo, seguro estoy con mi gato

<br/>

### 🛠️ Proyectos destacados

<details open>
<summary><b>🎓 SIBIA — Sistema Inteligente de Bienestar e Inteligencia Académica</b></summary>
<br/>

Plataforma full-stack universitaria con detección de riesgo estudiantil basada en IA.
- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** Node.js + Express
- **Base de datos:** PostgreSQL (Supabase) — ~216,000 registros en 17 tablas
- **IA:** Claude API para predicción de riesgo con Explainable AI (XAI)
- **Características:** 7 roles RBAC, cuestionario de bienestar de 10 dimensiones, lógica de escalamiento de alertas

🔗 [Ver repositorio](https://github.com/Robert-29/SIBIA)
</details>

<details>
<summary><b>🌉 Puente Levadizo IoT</b></summary>
<br/>

Sistema de automatización de un puente levadizo con monitoreo en tiempo real.
- **Hardware:** ESP32, servomotores MG995, sensores IR, semáforos
- **Dashboard:** React + Vite + Supabase Realtime
- **Backend:** Node.js (ES Modules) + Supabase con procedimientos almacenados para estadísticas
</details>
